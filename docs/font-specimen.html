Orrery · type specimen

Rounded sans candidates for the dial numerals

The reference dial's numerals are monoline with rounded terminals — the round milling tool that cuts them into the titanium shapes the letterforms. Each mini dial below is 225 px with 26 px numerals — pixel-identical to the shipped hour_dial.png, and the small date strip is the true 11 px date-ring size. The middle set leans hand-painted, ordered subtle → stronger.
